Why This Section

This page shows 10 common HTML mistakes and their correct versions. Each row describes the error, shows the wrong code, and gives the correct HTML. This supports Lab 01 learning: semantic structure, navigation, headings, tables, links, images, forms, and accessibility (e.g. aria-current, labels, alt).

10 Common HTML Errors and Correct Answers

Common HTML errors and correct markup (Lab 01 scope)
# Error / Issue Wrong HTML Correct HTML
1 Link directly inside ul (missing li) 
<nav><ul>
  <a href="index.html">Home</a>
  <li><a href="cv.html">CV</a></li>
</ul></nav>
 
<nav aria-label="Primary navigation">
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="cv.html">CV</a></li>
  </ul>
</nav>
2 Label for does not match input id 
<label for="msg">Message</label>
<textarea id="message" name="message"></textarea>
 
<label for="message">Message</label>
<textarea id="message" name="message"></textarea>
3 Heading level skipped (h1 then h3) 
<h1>Page Title</h1>
<h3>Subsection</h3>
 
<h1>Page Title</h1>
<h2>Subsection</h2>
4 Table with no caption, no th/scope 
<table>
  <tr><td>Model</td><td>Predicts</td></tr>
  <tr><td>KLM</td><td>Time</td></tr>
</table>
 
<table>
  <caption>Models and predictions</caption>
  <thead>
    <tr><th scope="col">Model</th>
    <th scope="col">Predicts</th></tr>
  </thead>
  <tbody>
    <tr><th scope="row">KLM</th>
    <td>Time</td></tr>
  </tbody>
</table>
5 More than one link with aria-current="page" 
<li><a href="index.html" aria-current="page">Home</a></li>
<li><a href="cv.html" aria-current="page">CV</a></li>
 
<li><a href="index.html">Home</a></li>
<li><a href="cv.html" aria-current="page">CV</a></li>
6 Multiple h1 on one page 
<h1>Main Title</h1>
<p>...</p>
<h1>Another Title</h1>
 
<h1>Main Title</h1>
<p>...</p>
<h2>Another Section</h2>
7 Image without alt attribute 
<img src="photo.jpg">
 
<img src="photo.jpg" alt="Description of the image">
Use alt="" only for decorative images.
8 Vague link text (“Click here”) 
<a href="norman2klm.html">Click here</a>
 
<a href="norman2klm.html">Norman–GOMS–KLM comparison</a>
9 Form control without an associated label 
Email: <input type="email" name="email">
 
<label for="email">Email</label>
<input type="email" id="email" name="email">
10 Using only div for structure (no semantic elements) 
<div class="header"><h1>Title</h1></div>
<div class="nav">...</div>
<div class="main">...</div>
 
<header><h1>Title</h1></header>
<nav aria-label="Primary navigation">...</nav>
<main>...</main>

Summary

Avoiding these 10 errors improves clarity, maintainability, and accessibility. Lab 01 requires semantic elements (header, nav, main, section, article, footer), list-based navigation, one h1 and no heading skips, proper tables with caption and th/scope, descriptive links, alt on images, and correct use of aria-current and form labels.
